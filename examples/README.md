# Examples

Two minimal demos showing where `optical-center` plugs into a real
project:

- [`react-vite/`](./react-vite/) — Vite plugin in a React app, the
  `<svg opticalCenter>` JSX form.
- [`vanilla-html/`](./vanilla-html/) — CLI driven, takes a folder of
  SVGs and writes perceptually centered copies next to it.

Each subfolder has its own `README.md` with run instructions.
